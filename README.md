# Intelligent Business Analytics & Funnel Optimization Platform

### [🚀 View Live Demonstration](https://build-by-ayush-intelligent-business-analytics-and-fu-app-ndnyxy.streamlit.app/)

## 🌟 Project Overview
This platform is a comprehensive **Decision Intelligence System** designed to move businesses from simple reporting to active diagnostics. While standard dashboards typically show only outcomes, this system analyzes the entire customer journey to identify where value is created and where it is lost.

The project integrates user behavior tracking, transactional data, and post-purchase logistics into a single, cohesive narrative. It answers the fundamental question: **Which products and behaviors drive sustainable value after accounting for real-world friction?**

---

## 🛠️ Core Capabilities

### 1. Funnel & Behavioral Intelligence
The system tracks the "ideal customer journey" through six critical stages:  
`Visit → Search → View → Add to Cart → Checkout → Purchase`

- **Drop-off Diagnostics**: Identifies the exact stage where the highest number of users leave the platform  
- **Device & Segment Analysis**: Compares conversion rates across Mobile, Desktop, and Tablet to pinpoint UX-specific issues  
- **Friction Measurement**: Uses behavioral friction analysis to calculate the median time users hesitate between steps, revealing hidden confusion or pricing doubts  

### 2. Post-Purchase & Return Diagnostics
Returns are treated as a financial and operational problem rather than just a support task.

- **Revenue Leakage**: Tracks **Refund Rates** and **Net Revenue After Returns** to show the actual realized value of sales  
- **Root Cause Mapping**: Correlates return reasons (like "Fit/Size" or "Damage") with specific product categories using advanced heatmaps  
- **Operational Efficiency**: Measures the delay in return processing to identify friction in internal approval workflows or logistics  

### 3. Strategic Product Health
The platform moves beyond "top-selling" lists to evaluate products based on their total impact on the bottom line.

- **Profit Allocation Logic**: Distributes order-level profit proportionally across individual items for accurate SKU-level tracking  
- **The 4-Quadrant Matrix**: Categorizes products into:
  - **Stars** (High Revenue / High Profit)  
  - **Hidden Gems** (High Profit / Low Revenue)  
  - **Inefficient Products** (High Revenue / Low Profit)  
- **Target Achievement**: Compares current monthly performance against historical averages to determine whether a product is in a "Growth Zone"  

### 4. The Automated Insight Engine
The analytical core of the system uses a rule-based inference engine to monitor data and surface meaningful insights.

- **Critical Alerts**: Flags sharp declines in demand or sudden spikes in return rates  
- **Narrative Generation**: Converts complex patterns into clear business insights, such as identifying products that become loss-making after returns

## 📈 Why This Project is Impactful

- **Eliminates Manual Interpretation**: Surfaces the most critical insights without requiring manual analysis of multiple dashboards  
- **Financial Integrity**: Prevents misleading conclusions by accounting for returns and true profit contribution  
- **Decision Support**: Guides strategy on product scaling, discontinuation, and checkout optimization  

---

## 🏗️ System Architecture

- **Frontend**: Interactive UI with custom stat cards and responsive layouts  
- **Data Layer**: Multi-table relational schema including `events`, `transactions`, `order_items`, `products`, and `returns`  
- **Technical Rigor**: Includes defensive engineering (e.g., `safe_divide`) and performance optimizations like caching

## 🚀 How to Use the Platform

1. **Overview**: Review overall business health and regional performance  
2. **Analyze**: Explore Funnel and Returns modules to identify friction points  
3. **Optimize**: Use Product Analysis to refine pricing and catalog strategy  
4. **Act**: Leverage Key Signals and Insight Engine outputs for data-driven decisions
